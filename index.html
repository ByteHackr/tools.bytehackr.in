Quick Reference 

                    

                        

                            

                                
Anchors

                                ^ Start of string

                                $ End of string

                                \b Word boundary
                            

                            

                                
Character Classes

                                \d Digit [0-9]

                                \w Word [a-zA-Z0-9_]

                                \s Whitespace

                                . Any character
                            

                            

                                
Quantifiers

                                * 0 or more

                                + 1 or more

                                ? 0 or 1

                                {n,m} n to m times
                            

                            

                                
Groups

                                (abc) Capture group

                                (?:abc) Non-capture

                                (?=abc) Lookahead

                                (?!abc) Neg. lookahead
                            

                            

                                
Flags

                                g Global

                                i Case insensitive

                                m Multiline

                                s Dotall
                            

                            

                                
Special

                                [abc] Character set

                                [^abc] Negated set

                                a|b Alternation

                                \ Escape
